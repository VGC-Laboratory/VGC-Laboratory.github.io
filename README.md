# VGC Laboratory VPM Repository

VGC (Virtual Gear Creation) による、Unityでのアバター制作・デバッグを効率化するためのツール配布用リポジトリです。

## インストール方法 (VCC)

以下のリンクをクリックして、VCCにリポジトリを追加してください。一度追加すれば、VGCが提供するすべてのツールをVCC上からインストール・管理できるようになります。

[👉 VCCに VGC Laboratory リポジトリを追加する](vcc://vpm/addRepo?url=https://vgc-laboratory.github.io/vpm.json)

追加後、各ツールの「Manage Project」から必要なツールをインストールしてください。

---

## 公開中のツール

### 🔍 Missing Checker
Unityシーン内のMissing参照（壊れたリンク）や、マテリアルの空スロットを検知するツールです。

---

## 使い方

### Missing Checker

1.  Unityメニューの `VGC > Debug Tools > Missing Checker` を選択してウィンドウを開きます。
2.  「Missing参照をスキャン」ボタンをクリックします。
3.  検出された項目がリスト表示されます。
    -   **選択ボタン**: クリックすると、該当するGameObjectがHierarchy上でハイライトされます。
    -   **アイコン表示**: どのコンポーネントのどのプロパティがMissingなのか一目で確認できます。
